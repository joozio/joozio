# Hey, I'm Pawel

I build things — mostly at the intersection of AI, automation, and e-commerce.

By day, I lead e-commerce operations. By night, I build tools that make my life (and hopefully yours) easier.

## What I'm working on

- **[Digital Thoughts](https://thoughts.jock.pl)** — My blog about building with AI, automation, and the messy reality of shipping things.
- **Wiz** — My AI automation framework. An agent system that handles everything from job searches to deployment pipelines. Open-sourcing parts of it soon.

## Open Source

- **[Dungeon of Opus](https://github.com/joozio/dungeon-of-opus)** — A complete roguelike built by Claude Opus 4.6 in one file. Procedural dungeons, 7 enemy types, boss fight. [Play it](https://wiz.jock.pl/experiments/dungeon-of-opus)
- **[Agent Arena](https://github.com/joozio/agent-arena)** — Prompt injection security testing tool. 10 hidden attacks, three-state scoring, leaderboard. [Try it](https://wiz.jock.pl/experiments/agent-arena)
- **[Agent Orchestra](https://github.com/joozio/agent-orchestra)** — Multi-agent coordination visualizer. Canvas particles, task dependencies, message passing. [See it](https://wiz.jock.pl/experiments/agent-orchestra)
- **[Claude Skill Auditor](https://github.com/joozio/claude-skill-auditor)** — Security scanner for Claude Code skills. Detects malicious patterns, risk scoring, auto-install gating.

## My approach

I believe in building over planning, shipping over polishing, and being honest about what works and what doesn't. Most of what I write comes from things I actually tried — including the failures.
